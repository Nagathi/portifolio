Primeiro é necessário definir as tecnologias utilizadas nesse projeto que foram Angular com TypeScript, Spring Boot com Java 17 e MySQL. Nesse projeto foi construída uma página estática que renderiza diversos componentes implementando o conceito de single-page application, é um projeto ambicioso que visa implementar uma plataforma para depositar gráficos táteis, essa plataforma já existe e esse projeto é uma réplica dela implementando tecnologias diferentes e um código feito do 0. No que diz respeito ao Frontend, essa página implementa cadastro de usuários com dados e uma foto que pode ser alterada, como se fosse um perfil de rede social, cadastro de arquivos e dados desses arquivos. No Backend foram implementadas rotas que aceitam requisições para cadastrar arquivo, usuário, fazer alterações nos arquivos e nos usuários e outros métodos utilizando o design pattern MVC padrão do Spring Boot com a camada de modelo de negócios Service.
